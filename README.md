# Yunzai-bot-useless-plugins
## 让我帮你百度一下(Let me Baidu it for you)
还在为伸手党而烦恼吗(
还不知道如何反击伸手党吗(
装上此插件将为你解决你的烦恼,自动回怼伸手党,让他们自己学会如何百度(bushi


安装方法:将"让我帮你百度一下.js"文件放进example文件夹即可

使用方法:在机器人所在的群聊中发送 百度+问题 即可
